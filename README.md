# Asp.Net-Core-Web-API-and-Angular-7-CRUD

## Description :
  This project is about "Banking Card's System " with the CRUD(create,read,update, delete) operations.

## CRUD Operations - 
Insert, update, delete and retrieve are implemented in Asp.Net Core Web API with Angular 7. First of all, Clone and set a Web API project in Asp.Net Core with required methods at server side using Entity Framework Core and SQL Server DB. Then Angular 7 Project consumes those methods from the client side.

## Tools Used : 
              VS Code, Visual Studio, SSMS(SQL Server), Postman.
